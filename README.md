# ⚔️ Ender Tiers

**The definitive ranking system for elite Minecraft PvP.**

Ender Tiers is a high-performance web platform designed to track, categorize, and display player skill levels (Tiers) across the Ender network. Built with a "Void & Amethyst" aesthetic, it provides a sleek, competitive alternative to traditional ranking sites.

![Java](https://img.shields.io/badge/Java-21-orange)
![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3.4-green)

---

## 🚀 Features
*   **Vanguard Leaderboard:** Real-time Elo tracking with automated tier assignment.
*   **Glassmorphism UI:** A modern, blur-heavy dark theme optimized for gamers.
*   **Secure API:** v1 REST API for fetching player data into Minecraft plugins.
*   **Responsive Design:** Fully functional on mobile and desktop devices.

---

## 🛠️ Tech Stack
*   **Backend:** Java 21 / Spring Boot 3.4
*   **Database:** MySQL (JPA/Hibernate)
*   **Frontend:** HTML5 / Tailwind CSS
*   **Data Source:** PaperMC 1.21.11 API
